Monte Carlo retirement sketchbook

Growthorama

A small browser app that runs Python in the page, simulates 100 market paths, and translates the results into real-dollar withdrawals, retirement balance, and what might be left behind.

Inputs

Everything is shown in today's dollars. The withdrawal rule applies the same percentage every year to that year's portfolio value.

  • The model uses 100 Monte Carlo paths with annual returns drawn from a normal distribution.
  • Returns are nominal, then converted back to today's dollars using the inflation assumption.
  • The shaded band uses empirical Monte Carlo percentile bounds matching -1σ and +1σ, not standard deviation.
  • Annual inputs are added only before retirement. Once withdrawals begin, contributions stop.
