Portfolio withdrawal sketchbook

Growthorama

A small browser app that runs Python in the page, projects a portfolio with and without management fees, and translates the result into real-dollar withdrawals, value at the start of withdrawals, and what might still be left at the maximum age.

Inputs

Everything is shown in today's dollars. The withdrawal rule applies the same percentage every year to that year's portfolio value.

Loading Python runtime...
  • The model uses one deterministic path based on the average market return you enter.
  • Returns are nominal, then converted back to today's dollars using the inflation assumption.
  • Management fees are applied every year after growth, and every result is shown both with fees and without fees.
  • Annual inputs are added only before withdrawals begin. Once withdrawals begin, contributions stop.
